Featured Tools
Interactive Smith Chart
Click anywhere on the chart to read off impedance, return loss, and VSWR. Plot multiple impedance points and visualise them against a full resistance/reactance grid.
SPICE Simulator
Real SPICE engine in your browser using Modified Nodal Analysis. Run .AC, .DC, .OP, and .TRAN analyses with R, L, C, V, and I elements — no installs required.
RF Circuit Simulator
Build ladder networks from R, L, C, and transmission line elements. Computes S-parameters, impedance, phase, group delay, and VSWR vs frequency using ABCD matrix analysis.
Link Budget Calculator
End-to-end RF link analysis: TX power, path loss, noise floor, and SNR margin with colour-coded pass/fail output.
Multi-Turn Coil Designer
Wheeler formula inductance, tuning capacitor, wire resistance, and Q factor for helical RF coils — designed especially for MRI tuning.
LC Filter Designer
Generate Butterworth or Chebyshev LC ladder component values up to 9th order. Low-pass or high-pass, with full bill of materials.
