S-Parameter Plotter

Plots S-parameters from Touchstone files (.s1p–.s4p). Supports magnitude, phase, real/imaginary, and dB formats. Automatically detects file format (MA, DB, RI). Useful for visualising VNA measurement data.

Equations & Parameters ▸
.s1p1-port network: S11 only.
.s2p2-port: S11, S21, S12, S22. Most common for amplifiers and filters.
.s3p/.s4p3 and 4-port networks (e.g. directional couplers, power dividers).
FormatMA = magnitude/angle, DB = dB/angle, RI = real/imaginary. Auto-detected.
Reference: Touchstone® format: Keysight EEsof EDA, Touchstone File Format Specification rev. 1.1 (2002)
