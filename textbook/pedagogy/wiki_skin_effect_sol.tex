\begin{enumerate}[leftmargin=1.6em,itemsep=2pt]
\item $\delta \propto 1/\sqrt{f}$, so lowering $f$ by $10\times$ raises $\delta$ by $\sqrt{10}$: $2.1\times3.16 = 6.6\ \mu\mathrm{m}$.
\item It falls by $\sqrt{100} = 10\times$.
\item $\delta_{\text{Au}} = \delta_{\text{Cu}}\sqrt{\sigma_{\text{Cu}}/\sigma_{\text{Au}}} = 2.1\sqrt{5.8/4.1} = 2.5\ \mu\mathrm{m}$; a plating a few skin depths thick ($\sim$5--8~$\mu$m) carries essentially all the current.
\end{enumerate}
