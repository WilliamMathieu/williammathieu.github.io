\begin{workedexample}[Worked Example: A 75~$\Omega$ load on a 50~$\Omega$ line]
For $Z_L = 75\ \Omega$ on $Z_0 = 50\ \Omega$,
$\Gamma = (75-50)/(75+50) = 0.20$, hence
\[ \text{VSWR} = \frac{1+|\Gamma|}{1-|\Gamma|} = \frac{1.2}{0.8} = 1.5, \qquad
   \text{RL} = -20\log_{10}(0.2) = 14.0\ \text{dB}. \]
The reflected power is $|\Gamma|^2 = 4\%$ and the mismatch loss is
$-10\log_{10}(1-0.04) = 0.18\ \text{dB}$.
\end{workedexample}

\begin{keytakeaways}
\begin{itemize}[leftmargin=1.4em,itemsep=2pt]
\item $|\Gamma|$, VSWR, RL, and mismatch loss are one physical fact in four algebras.
\item Reflected-power fraction $= |\Gamma|^2$; $\text{VSWR} = (1+|\Gamma|)/(1-|\Gamma|)$.
\item RL $\ge 10\ \text{dB}$ (VSWR $\le 1.9$) is a common practical floor.
\end{itemize}
\end{keytakeaways}

\item A return loss of $20\ \text{dB}$ corresponds to what $|\Gamma|$ and VSWR?
\item What VSWR and return loss does a perfectly matched load have?
\end{exercises}
