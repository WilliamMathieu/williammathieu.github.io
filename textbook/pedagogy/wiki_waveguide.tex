\begin{workedexample}[Worked Example: WR-90 cutoff]
The dominant $TE_{10}$ cutoff depends only on the broad wall $a$:
$f_c = c/(2a)$. WR-90 has $a = 22.86\ \text{mm}$, so
\[ f_c = \frac{3\times10^{8}}{2(22.86\times10^{-3})} = 6.56\ \text{GHz}. \]
Single-mode operation runs from $f_c$ to about $2f_c = 13.1\ \text{GHz}$; the
standard X-band rating $8.2$--$12.4\ \text{GHz}$ sits inside that window.
\end{workedexample}

\begin{keytakeaways}
\begin{itemize}[leftmargin=1.4em,itemsep=2pt]
\item $TE_{10}$ cutoff $f_c = c/(2a)$; below it the guide will not propagate.
\item Useful single-mode band is roughly $f_c$ to $2f_c$.
\item Waveguide is low-loss and high-power but bulky and band-limited.
\end{itemize}
\end{keytakeaways}

\begin{exercises}
\item For $a = 15\ \text{mm}$, find the $TE_{10}$ cutoff.
\item Can a $5\ \text{GHz}$ signal propagate in WR-90 ($f_c = 6.56\ \text{GHz}$)?
\item Roughly where is the top of WR-90's single-mode band?
\end{exercises}
