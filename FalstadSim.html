Circuit Simulator (Falstad)

Full interactive circuit simulator by Paul Falstad, embedded in the page. Real-time animated current flow, oscilloscope probes, and a library of built-in components. Use the Draw menu to add components, right-click to edit values, and press Space to run or pause.

LOAD A PRESET CIRCUIT
How to load a preset circuit:
  1. Click a preset button above — the circuit text will appear below
  2. Click Copy Circuit Text
  3. In the simulator: File menu → Import From Text
  4. Paste the copied text and click OK
