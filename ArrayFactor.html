When multiple antenna elements are combined in a phased array, their signals add constructively in some directions and destructively in others, forming a radiation pattern with a main beam and sidelobes. The array factor (AF) describes this directivity pattern independent of the element type. By adjusting the steering angle θ₀, the main beam can be electronically steered without physically moving the antenna.

NNumber of elements in the linear array.
dElement spacing in wavelengths. λ/2 spacing (d=0.5) is standard — avoids grating lobes.
θ₀Steering angle (degrees). 0° = broadside (perpendicular to array axis).
AF(θ)Array factor vs angle. The pattern shows the main beam, sidelobes, and nulls.

Antenna Array Factor

A phased array combines multiple antenna elements whose signals add constructively in the steered direction and destructively elsewhere. The array factor describes this directivity pattern. λ/2 spacing (d = 0.5) is standard to avoid grating lobes.

NNumber of array elements.
dElement spacing in wavelengths. d = 0.5 is the standard choice.
θ₀Steering angle (degrees from broadside). 0° = perpendicular to array axis.
