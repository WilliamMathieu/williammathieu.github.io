Antenna Array Factor Plotter

The array factor is the radiation pattern contribution from the geometry and phasing of an antenna array — independent of the individual element pattern. Narrower beamwidth and higher directivity come from adding more elements. Use this to visualise steering, grating lobes, and pattern shaping.

Parameter reference ▸
NNumber of array elements.
dElement spacing in wavelengths (λ). Half-wavelength spacing (0.5λ) avoids grating lobes.
θ₀Main beam steering angle (°). 0° = broadside, ±90° = end-fire.
Inputs
λ
0.5 avoids grating lobes
