VSWR / Return Loss / Reflection Coefficient Converter

Enter any one of the four equivalent match-quality metrics and all others update instantly. These four quantities are all different ways of expressing the same physical thing — how well two impedances are matched.

Parameter reference ▸
VSWRVoltage Standing Wave Ratio = (1+|Γ|)/(1−|Γ|). Perfect match = 1. Typically target ≤ 2.
|Γ|Reflection coefficient magnitude 0–1. 0 = perfect match, 1 = total reflection.
RLReturn loss = −20·log₁₀|Γ| dB. Higher = better. Common target > 10 dB.
MLMismatch loss = −10·log₁₀(1−|Γ|²) dB. Power lost to reflection.
